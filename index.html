Welcome to TennisForFans.com!

This site contains ATP's top 50 players' current ranking and their rank history starting from 2003. It also includes a map that shows all professional-level tournaments occurred in 2013. Top players' overall win-loss chart and grand-slam win-loss chart are also neatly presented.


Credit

Charts on this site use data collected from ATP's homepage and Wikipedia. For further information about each player and tournament, please visit them directly.

This website uses jQuery, D3, NVD3, MiddleMan, jqPlot etc.

The data collected from ATP are stored in Cloudant.

Both this website and its source code are hosted on github, please feel free to clone and add functionalities you want.
